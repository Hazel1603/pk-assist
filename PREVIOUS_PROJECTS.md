## Project 1: AI CLI Assistant

I built a command-line AI assistant that can:

- use the OpenAI API
- keep conversation history
- call a filesystem tool
- read local text files
- summarize files
- answer questions about files
- return structured responses
- log token usage
- stream response events internally

## Concepts Covered

- conversation history

- error handling
